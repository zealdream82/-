# Mealjeop (밀접) Landing Page

식당 키오스크 맞춤 영양제 서비스 '밀접'의 랜딩 페이지입니다.

## 업데이트 내역
- 백엔드 챗봇 기능 제거 및 [고객 문의 폼] 교체 후 프론트엔드 최적화
- **긴급 수정:** 업로드된 이미지 파일이 확장자는 `.png`이나 실제로는 `JPEG` 파일이었음. 이로 인해 Netlify(웹 브라우저)에서 엑스박스가 뜨던 치명적인 문제 파악 완료 
  - 파일명을 `hero.jpg`로 수정하여 확장자와 포맷을 일치시킴으로써 엑스박스 완전 해결
- 이미지 경로를 public 폴더 우선으로 재설정
