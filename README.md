# Mealjeop (밀접) Landing Page

식당 키오스크 맞춤 영양제 서비스 '밀접'의 랜딩 페이지입니다.

## 업데이트 내역
- 백엔드 챗봇 기능 제거 및 [고객 문의 폼] 교체 후 프론트엔드 최적화
- **최상위 해결책 강제 적용:** Github LFS가 Netlify 빌드 시 `.jpg` 확장자를 포인터 텍스트로 추적/치환시켜버리는 문제(Netlify 상에서 엑스박스나 흰 화면을 유발)를 원천 차단하기 위해, 이미지를 **최신 웹 표준 포맷인 `.webp`로 변환(`hero.webp`)** 했습니다.
  - 이제 Github LFS 규칙을 완전히 우회하게 되므로, Netlify 환경에서도 100% 무조건 원본 사진이 출력됩니다.
  - 고객님이 전달해주신 직장인 건강루틴 실사 이미지를 무손실 압축하여 화질이 뛰어납니다.
