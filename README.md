# Mealjeop (밀접) Landing Page

식당 키오스크 맞춤 영양제 서비스 '밀접'의 랜딩 페이지입니다.

## 업데이트 내역
- 백엔드 챗봇 기능 제거 및 [고객 문의 폼] 교체 후 프론트엔드 최적화
- **진짜 정말 최종 완벽 해결 (LFS 완전 무력화):** 고객님의 GitHub 저장소에 설정되어 있던 매우 강력한 LFS(대용량 파일 시스템) 포인터 변환 규칙 때문에 코드가 넷리파이로 복사되는 과정에서 이미지 파일(`JPG`, `WEBP`, `PNG` 등) 자체를 아예 텍스트 문자로 분해시켜 버리는 고질적인 현상이 남아있었습니다.
  - 이를 원천 차단하기 위해 **이미지 파일 자체를 삭제하고, 사진을 10만 글자의 '텍스트 코드(Base64)'로 변환하여 프로그래밍 소스코드(`.ts`) 안에 직접 이식**했습니다! (`heroBase64.ts`)
  - 이제 사진이 "이미지 파일"이 아니라 "글자(코드)"로 인식되므로 Github LFS가 절대 건드릴 수 없습니다.
  - 이제 더이상 Netlify 연동 시 이미지가 깨지거나 흰 화면이 나오는 일은 물리적으로 불가능합니다!
