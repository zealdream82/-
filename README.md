# Mealjeop (밀접) Landing Page

식당 키오스크 맞춤 영양제 서비스 '밀접'의 랜딩 페이지입니다.

## 업데이트 내역
- 백엔드 챗봇 기능 제거 및 [고객 문의 폼] 교체 후 프론트엔드 최적화
- **진짜 최종 해결:** Github LFS(대용량 파일 시스템) 제한 문제로 밝혀짐. 원본 이미지 크기(2MB)가 너무 커서 넷리파이가 Github에서 텍스트 조각만 받아오던 구조적 결함을 발견함.
  - 화질을 최대한 유지하며 이미지 용량을 50KB로 초고도로 압축 최적화 (`hero_optimized.jpg`)
  - 다시 `import` 방식으로 코드를 변경하여 빌드 시 무조건 포함되도록 100% 강제 처리
