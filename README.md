# Mealjeop (밀접) Landing Page

식당 키오스크 맞춤 영양제 서비스 '밀접'의 랜딩 페이지입니다.

## 업데이트 내역
- 백엔드 챗봇 기능 제거 및 [고객 문의 폼] 교체 후 프론트엔드 최적화
- **최종 종결 (메모리 변환 기술 적용):** LFS가 파일 시스템 레벨에서 개입하여 `.jpg`, `.webp` 등의 파일로 저장되는 모든 이미지를 포인터 텍스트로 오염시키는 근본적 원인을 차단했습니다.
  - 디스크에 이미지로 저장하는 과정을 아예 생략하고, 메모리(RAM) 상에서 직접 사진 데이터를 코드로 즉시 변환하여 `.ts` 파일 안에 삽입했습니다.
  - 이제 더 이상 "깨진 텍스트"가 이미지를 대체하는 현상이 발생할 수 없습니다. 완벽하게 성공적으로 넷리파이에 배포될 것입니다!
